Open the Dev Console for use

Table size is 5 x 5 positions.

Robot Commands:

  • place(x,y,f)
  • move()
  • left()
  • right()
  • report()
The robot toy must first be 'placed' on the table using the 'place' command.

place( x, y, f )

  • 'x + y' must be within 0 - 5
  • 'f' must be choice of four cardinal points e.g. "north", "south", "east", "west"
